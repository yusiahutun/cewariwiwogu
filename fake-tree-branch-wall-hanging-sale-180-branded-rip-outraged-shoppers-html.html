Fake tree branch wall hanging that is on sale for 180 is branded a rip off by outraged shoppers

September 2024 · 2 minute read

A fake tree branch that costs an eye-watering £180 has been branded a rip off by outraged shoppers.

Crate and Barrel is flogging the large, imitation conifer twig online for Christmas.

The branch has string tied on each end allowing it to be turned into a wall decoration.

The Hanging Branch Decor has just gone on sale online for £110, but shipping costs from the American store adds another £70.

The £110 branch costs £70 to ship to the UK. It has string tied on each end allowing it to be turned into a wall decoration

Its website reads: 'Modern yet rustic, this swag suspends a lifelike evergreen bough from a length of sturdy jute twine.

'The hanging branch harkens back to the earliest Scandinavian winter and Christmas decor, with a scattering of tiny battery-powered lights adding high-tech sparkle.
